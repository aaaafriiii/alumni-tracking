# Alumni Tracking System

## Deskripsi Project

Alumni Tracking System merupakan aplikasi web yang digunakan untuk melakukan pelacakan alumni melalui berbagai platform media sosial seperti LinkedIn, Instagram, Facebook, Google Scholar, dan ResearchGate. Sistem ini membantu admin dalam mengumpulkan informasi terkait aktivitas dan profil alumni secara lebih terstruktur.

Aplikasi ini dibuat sebagai bagian dari tugas Daily Project pada mata kuliah Rekayasa Perangkat Lunak.

---

## Fitur Sistem

Beberapa fitur utama pada sistem ini antara lain:

* Melihat daftar alumni
* Memilih nama alumni
* Melihat profil alumni

---

## Teknologi yang Digunakan

Berikut teknologi yang digunakan dalam pengembangan sistem:

* HTML
* CSS

* Node.js
